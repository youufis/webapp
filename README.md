网站内容管理系统<br>
一、实现内容：<br>
1、django默认自带后台（管理端帐号和密码：root/root）
2、后台自行添加修改删除类别和内容<br>
3、管理员和普通用户可以发布内容，普通用户需后台审核才能前台显示<br>
4、登录注册<br>
二、所需环境：<br>
  python3.7<br>
  Django=3.13<br>
  DjangoUeditor=1.8.143<br>
  six=1.15.0 <br>
三、注意事项<br>
1、使用默认数据库sqlite3<br>

四、运行测试：<br>
在项目目录下运行：<br>
webapp>python manage.py runserver:0.0.0.0:8000<br>
打开浏览器：127.0.0.1:8000<br>
